Бестиарий

Инструменты разработчика:

  • gulp - Таск менеджер
  • gulp-postcss - Постпроцессор
  • postcss-preset-env - набор плагинов для заданного окружения
  • postcss-import - пример "нестандартного" плагина
  • postcss-rtl - пример плагина-генератора кода

Бестиарий

Полезное:

  • stylelint - твой личный ревьювер стилей
  • gulp-sourcemap - карты исходного кода
  • gulp-csso - минификация стилей

Конфиг PostCSS


    const postcssOptions = [
        atImport(), postcssPresetEnv({
            stage: 0,
            autoprefixer: {
